## ¿Qué es APPSeguimiento?

**APPSeguimiento** es una aplicación web desarrollada con **Laravel 12** para gestionar el seguimiento de aprendices, instructores, fichas de caracterización y bitácoras en el contexto del SENA (Servicio Nacional de Aprendizaje de Colombia).

El sistema centraliza la información de formación profesional, permitiendo a instructores y coordinadores registrar y consultar datos de aprendices, asignar fichas de caracterización a programas y centros de formación, y llevar un registro documental de bitácoras en PDF.

## Características clave

- **Autenticación segura** — Login con correo y contraseña, sesiones protegidas con middleware `auth`
- **Búsqueda paginada** — Filtrado en tiempo real por nombre, número de documento y código en todos los módulos principales
- **Notificaciones por correo** — Alertas automáticas al crear, actualizar o eliminar registros de aprendices
- **Carga de archivos** — Bitácoras en PDF almacenadas en el sistema de archivos con rutas persistentes
- **CRUD completo** — Crear, leer, actualizar y eliminar en todos los módulos con validaciones del lado servidor

## Tecnologías utilizadas

| Componente | Tecnología |
|---|---|
| Framework | Laravel 12 (PHP 8.2+) |
| Base de datos | SQLite (configurable a MySQL/PostgreSQL) |
| Frontend | Blade Templates + Vite |
| Autenticación | Laravel Auth (sesiones) |
| Almacenamiento | Laravel Storage (disco público) |
| Correo | Laravel Mail (configurable) |
| Testing | PHPUnit 11 |

## Empezar

<Steps>
  <Step title="Instala las dependencias">
    Clona el repositorio y ejecuta `composer install` seguido de `npm install`.
  </Step>
  <Step title="Configura el entorno">
    Copia `.env.example` a `.env` y genera la clave de aplicación con `php artisan key:generate`.
  </Step>
  <Step title="Ejecuta las migraciones">
    Crea la base de datos y las tablas con `php artisan migrate`.
  </Step>
  <Step title="Inicia el servidor">
    Levanta el servidor de desarrollo con `php artisan serve` y accede en `http://localhost:8000`.
  </Step>
</Steps>

<Note>
  Consulta la [guía de instalación completa](/setup/installation) para configurar el entorno de producción y las variables de correo electrónico.
</Note>
