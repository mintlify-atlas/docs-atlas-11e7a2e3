**APPSeguimiento** es una aplicación web desarrollada con **Laravel 12** para gestionar el seguimiento académico y administrativo de aprendices, instructores, fichas de caracterización y bitácoras en el contexto del **SENA** (Servicio Nacional de Aprendizaje de Colombia).

El sistema centraliza la información de formación profesional en una sola plataforma, eliminando el uso de hojas de cálculo dispersas y permitiendo trazabilidad completa de los registros mediante notificaciones automáticas por correo electrónico.

## Pila tecnológica

| Componente | Tecnología | Versión |
|---|---|---|
| Framework backend | Laravel | 12.x |
| Lenguaje | PHP | 8.2+ |
| Base de datos | SQLite (desarrollo) / MySQL / PostgreSQL | — |
| Plantillas frontend | Blade Templates | — |
| Compilador de activos | Vite | 7.x |
| CSS | Tailwind CSS | 4.x |
| Autenticación | Laravel Auth (sesiones) | — |
| Almacenamiento de archivos | Laravel Storage (disco público) | — |
| Correo electrónico | Laravel Mail (SMTP configurable) | — |
| Cola de trabajos | Laravel Queue (driver database) | — |
| Testing | PHPUnit | 11.x |
| Inspector de logs | Laravel Pail | 1.2+ |

<Note>
  APPSeguimiento utiliza **SQLite** como base de datos predeterminada para facilitar la instalación en entornos de desarrollo. Para producción se recomienda cambiar a **MySQL** o **PostgreSQL**. Consulta la [configuración de base de datos](/setup/database) para más detalles.
</Note>

## Estructura del proyecto

El proyecto sigue la arquitectura estándar de Laravel con controladores MVC, modelos Eloquent y vistas Blade. Los módulos principales se organizan así:

- `app/Http/Controllers/` — Controladores para cada módulo (aprendices, instructores, bitácoras, fichas, etc.)
- `app/Models/` — Modelos Eloquent con relaciones entre tablas
- `app/Mail/` — Clase `AprendizMail` para el envío de notificaciones
- `resources/views/` — Plantillas Blade para la interfaz de usuario
- `database/migrations/` — Migraciones de base de datos
- `routes/web.php` — Definición de rutas de la aplicación
