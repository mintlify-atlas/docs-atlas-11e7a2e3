title: "Modelos de Datos"
description: "Referencia de modelos Eloquent, tablas de base de datos y relaciones en APPSeguimiento"
---

Todos los modelos de APPSeguimiento extienden `Illuminate\Database\Eloquent\Model` y usan `Nis` como clave primaria (en lugar del estándar `id`).

<Note>
La clave primaria en todos los modelos es `Nis` (Número de Identificación del Sistema), un entero autoincremental.
</Note>

<AccordionGroup>

<Accordion title="aprendices — tblaprendices">

| Propiedad | Valor |
|-----------|-------|
| Tabla | `tblaprendices` |
| Clave primaria | `Nis` |
| Timestamps | No |

**Campos fillable:**

| Campo | Descripción |
|-------|-------------|
| `tbltiposdocumentos_Nis` | FK → tiposdocumentos |
| `NumDoc` | Número de documento |
| `Nombres` | Nombres del aprendiz |
| `Apellidos` | Apellidos del aprendiz |
| `Direccion` | Dirección de residencia |
| `Telefono` | Teléfono de contacto |
| `CorreoInstitucional` | Correo institucional SENA |
| `CorreoPersonal` | Correo personal |
| `Sexo` | 1 = Masculino, 2 = Femenino |
| `FechaNac` | Fecha de nacimiento |
| `tbleps_Nis` | FK → eps |

**Relaciones:**

```php
public function tipodocumento()
{
    return $this->belongsTo(TiposDocumentos::class, 'tbltiposdocumentos_Nis', 'Nis');
}

public function eps()
{
    return $this->belongsTo(Eps::class, 'tbleps_Nis', 'Nis');
}
```

**Accessor:**

```php
public function getSexoTextoAttribute()
{
    return match ($this->Sexo) {
        1 => 'Masculino',
        2 => 'Femenino',
        default => 'No definido'
    };
}
```

Accede al valor legible con `$aprendiz->sexo_texto`.

</Accordion>

<Accordion title="User — tblusuarios">

| Propiedad | Valor |
|-----------|-------|
| Tabla | `tblusuarios` |
| Clave primaria | `Nis` |
| Timestamps | No |

Extiende `Illuminate\Foundation\Auth\User as Authenticatable`.

**Campos fillable:**

| Campo | Descripción |
|-------|-------------|
| `Nis` | Identificador único |
| `Correo` | Dirección de correo (única) |
| `Contrasena` | Contraseña hasheada (hidden en serialización) |

**Métodos personalizados:**

```php
public function getAuthPassword()
{
    return $this->Contrasena;
}
```

Sobrescribe el método estándar de Laravel para usar el campo `Contrasena` en lugar de `password`.
