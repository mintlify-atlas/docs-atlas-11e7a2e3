title: "Inicio rápido"
description: "Despliega APPSeguimiento y registra tu primer aprendiz en minutos"
---

Esta guía te lleva desde cero hasta tener APPSeguimiento funcionando con el primer aprendiz registrado. El proceso completo toma menos de 10 minutos.

<Note>
  Asegúrate de tener instalados **PHP 8.2+**, **Composer**, **Node.js** y **npm** antes de comenzar. Consulta los [requisitos del sistema](/setup/installation#requisitos-del-sistema) para más detalles.
</Note>

<Steps>
  <Step title="Clonar el repositorio">
    Clona el repositorio de APPSeguimiento desde GitHub:

    ```bash
    git clone https://github.com/DavidTorresT/APPSeguimiento.git
    cd APPSeguimiento
    ```
  </Step>

  <Step title="Instalar dependencias PHP">
    Instala los paquetes de Composer, incluyendo Laravel 12 y sus dependencias:

    ```bash
    composer install
    ```

    <Tip>
      Puedes usar `composer setup` para ejecutar automáticamente todos los pasos de instalación (dependencias PHP, copia del `.env`, generación de clave, migraciones, dependencias JS y compilación de activos) en un solo comando.
    </Tip>
  </Step>

  <Step title="Instalar dependencias JavaScript">
    Instala Vite, Tailwind CSS y los demás paquetes de Node.js:

    ```bash
    npm install
    ```
  </Step>

  <Step title="Configurar el archivo de entorno">
    Copia el archivo de ejemplo y genera la clave de aplicación:

    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

    Esto crea el archivo `.env` con todos los valores predeterminados y genera un valor seguro para `APP_KEY`, que Laravel usa para cifrar sesiones y datos.

    <Warning>
      No omitas el paso `php artisan key:generate`. Sin `APP_KEY`, la aplicación no puede cifrar sesiones y fallará al iniciar sesión.
    </Warning>
  </Step>

  <Step title="Crear el archivo de base de datos y ejecutar migraciones">
    APPSeguimiento usa **SQLite** por defecto. Crea el archivo y ejecuta las migraciones:

    ```bash
    touch database/database.sqlite
    php artisan migrate
    ```

    Esto crea todas las tablas del sistema: `tblaprendices`, `tblinstructores`, `tblfichasdecaracterizacion`, `tblbitacoras` y más.
  </Step>

  <Step title="Compilar los activos frontend">
    Compila los archivos CSS y JavaScript con Vite:

    ```bash
    npm run build
    ```

    Esto procesa `resources/css/app.css` y `resources/js/app.js` generando los activos optimizados en `public/build/`.
  </Step>

  <Step title="Iniciar el servidor de desarrollo">
    Levanta el servidor de desarrollo de Laravel:

    ```bash
    php artisan serve
    ```

    La aplicación estará disponible en [http://localhost:8000](http://localhost:8000).

    <Tip>
      Para desarrollo con recarga en caliente, usa `composer dev` en lugar de `php artisan serve`. Este comando inicia simultáneamente el servidor PHP, el servidor de colas, el inspector de logs (Pail) y el servidor de desarrollo de Vite.
    </Tip>
  </Step>

  <Step title="Crear el primer usuario">
    Navega a [http://localhost:8000/usuarios/create](http://localhost:8000/usuarios/create) y completa el formulario de registro:

    | Campo | Descripción |
    |---|---|
    | **Correo** | Dirección de correo electrónico del usuario |
    | **Contraseña** | Contraseña de acceso al sistema |

    Este es el usuario administrador inicial con el que podrás iniciar sesión.
  </Step>

  <Step title="Iniciar sesión">
    Ve a [http://localhost:8000/login](http://localhost:8000/login) e ingresa con el correo y contraseña que acabas de crear.
  </Step>

  <Step title="Registrar el primer aprendiz">
    Navega a [http://localhost:8000/aprendices](http://localhost:8000/aprendices) y haz clic en **Crear aprendiz**.

    Completa los campos requeridos:

    | Campo | Tipo | Descripción |
    |---|---|---|
    | **NumDoc** | Texto | Número de documento de identidad |
    | **Nombres** | Texto | Nombres del aprendiz |
    | **Apellidos** | Texto | Apellidos del aprendiz |
    | **Direccion** | Texto | Dirección de residencia |
    | **Telefono** | Texto | Número de teléfono de contacto |
    | **CorreoInstitucional** | Email | Correo SENA del aprendiz |
    | **CorreoPersonal** | Email | Correo personal del aprendiz |
    | **Sexo** | Selección | Masculino (1) o Femenino (2) |
    | **FechaNac** | Fecha | Fecha de nacimiento |
    | **Tipo de documento** | Selección | Referencia a `tbltiposdocumentos` |
    | **EPS** | Selección | Entidad de salud, referencia a `tbleps` |

    <Note>
      Al guardar el registro, el sistema envía automáticamente una notificación por correo a la dirección configurada en `MAIL_FROM_ADDRESS`. Si aún no has configurado el correo, este envío se registrará en los logs de Laravel en `storage/logs/laravel.log`.
    </Note>
  </Step>
</Steps>
